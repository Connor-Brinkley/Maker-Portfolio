images/

This directory is for images used by the site. Add your own photos or GIFs here with these filenames (or update `index.md` to match your filenames):

- headshot.jpg — portrait or project hero image
- gait-demo.jpg — short motion / gait photo or GIF
- electronics.jpg — board or wiring photo

Keep images reasonably sized (recommended <= 1–2 MB each). Commit them to the repo and the site will show them on GitHub Pages.
